## Live Demo
Try it live here: [Text to Speech Converter](https://asma-ehsan.github.io/Text-to-Voice-Converter/)

## Features
- Convert any text input into speech instantly.
- Dropdown to select different voices available in your browser.
- Responsive, modern, and visually appealing interface.
- Smooth interaction with real-time voice changes.
- Built using Vanilla JavaScript and simple DOM manipulation.

## How to Use
1. Open `index.html` in your browser.
2. Type or paste text into the textarea.
3. Select a preferred voice from the dropdown menu.
4. Click the **Listen** button to hear the text read aloud.
